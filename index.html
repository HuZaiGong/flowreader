项目简介

为什么选择 FlowReader

在信息洪流中，FlowReader 为你打造一片宁静的阅读空间。纯本地设计，无需网络连接，随时随地享受阅读时光。

离线优先

完全离线的阅读体验，无需网络连接，所有数据本地存储，保护隐私的同时让你随时随地畅快阅读。

现代化 UI

基于 Jetpack Compose 构建的 Material 3 界面，支持动态配色（Material You），视觉体验精美流畅。

多格式支持

原生支持 EPUB、TXT、PDF 和 Markdown 格式的电子书，一键导入，即刻阅读，无需繁琐的格式转换。

纯净开源

采用 GPL-3.0 开源协议，代码完全透明公开。自由使用、修改和分发，社区驱动持续迭代。

核心功能

强大的阅读体验

从智能书库管理到个性化阅读设置，FlowReader 为每一位读者提供贴心的功能支持。

智能书库

基于 Room 数据库的高效书库管理，支持书籍分类、搜索、收藏和阅读进度自动保存。

多种阅读模式

支持日间/夜间模式切换，自定义字体大小、行间距和背景色，打造最舒适的阅读环境。

阅读统计

可视化阅读数据统计，包括阅读时长、每日进度和阅读趋势，帮助你建立良好的阅读习惯。

全文搜索

强大的全文搜索功能，快速定位书籍内容，支持关键词高亮显示，高效查找所需信息。

决策转盘

有趣的决策转盘功能，帮助你在不知道读哪本书时快速做出选择，增添阅读乐趣。

数据备份

支持数据备份和还原，通过本地备份文件保存书库和阅读进度，数据迁移更安心。

应用场景

随时随地阅读

无论是在通勤路上还是睡前时光，FlowReader 都是你的最佳阅读伴侣。

通勤路上

地铁、公交上的碎片时间，打开 FlowReader 继续未读完的章节，离线也能畅读。

睡前阅读

自动切换夜间模式，护眼配色与舒适排版，让阅读成为睡前最佳放松方式。

学习研究

PDF 论文与 EPUB 教材一键导入，全文搜索与高亮标注，学习效率倍增。

旅行途中

无需担心网络信号，所有书籍本地存储，长途旅行也能沉浸书海。
