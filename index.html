✨ 功能特性

全面的阅读功能

从书库管理到深度阅读，FlowReader 提供了完整的功能矩阵

📚

书库管理

导入本地书籍（EPUB、TXT、PDF、Markdown），支持批量导入、书架展示、分类浏览与排序管理[reference:0]。

🎨

阅读主题

浅色、深色、护眼、羊皮纸、AMOLED 纯黑五种主题，随心切换[reference:1]。

📝

笔记与批注

文字高亮划线，添加想法批注，支持黄、绿、蓝、粉、橙多种高亮颜色，笔记可导出[reference:2]。

🔍

全文搜索

单本书籍内全文检索，搜索结果高亮显示，快速跳转定位[reference:3]。

🗣️

文本朗读

接入系统 TTS 语音引擎，支持调节语速和音调，中英文语音识别[reference:4]。

🎲

决策转盘

可定制的决策转盘工具，支持自定义选项和颜色，加权随机算法，帮助解决阅读选择困难[reference:5]。

📊

阅读统计

每日阅读时长和页数统计，阅读进度自动记录，支持设置每日阅读目标[reference:6]。

🔖

书签功能

随时添加书签，书签列表管理，快速跳转书签位置[reference:7]。

💾

数据管理

书籍和阅读进度备份与恢复，阅读记录导出，纯本地运行无云端依赖[reference:8]。

📖 阅读体验

沉浸式阅读，随心定制

每一个细节都为更好的阅读体验而设计

📏 字体大小
12sp - 32sp 可调，适配不同阅读习惯[reference:9]
📐 行间距
1.0 - 2.5 可调，优化阅读舒适度[reference:10]

🌙 自动夜间模式

📑 PDF 支持
流畅阅读，支持缩放和拖拽翻页[reference:14]
🖐️ 边缘手势

🔆 屏幕常亮
阅读时保持屏幕常亮，无需频繁点击[reference:16]

切换章节自动恢复滚动位置[reference:17]
📊 阅读进度

⚡ 性能优化

流畅，不止于表面

多项优化技术，确保即使在低配置设备上也能流畅运行

🧠 智能内存管理

MemoryManager 实时监控内存压力，自动调节缓存策略[reference:19]

🗑️ LRU 缓存管理

CacheManager 实现 LRU 缓存，自动淘汰低优先级缓存[reference:20]

📚 章节内存缓存

智能缓存已读取章节，减少数据库查询，提升阅读切换速度[reference:21]

🌊 流式书籍解析

支持进度回调，大文件不再卡顿[reference:22]

🖼️ 图片缓存优化

内存 + 磁盘双缓存，滚动更流畅[reference:23]

⏱️ 进度延迟写入

3 秒 debounce 减少数据库写入，提升整体性能[reference:24]

📄 PDF 流式渲染

大型 PDF 文件流畅加载[reference:25]

⚡ Lazy Loading

章节内容按需加载，首屏加载更快[reference:26]

🔧 构建指南

快速构建 APK

环境要求：Android SDK 34+、Kotlin 1.9+、JDK 17[reference:31]

Terminal
# 克隆项目
git clone https://github.com/HuZaiGong/flowreader.git

# 进入项目目录
cd flowreader

# 构建 Debug APK
./gradlew assembleDebug

# 构建 Release APK
./gradlew assembleRelease
